# Agentic CMS Controller

Этот репозиторий содержит ИИ-навыки (Agent Skills) для автономного управления контентом и настройками клиентских сайтов, базирующихся на шаблоне Agentic CMS.

Архитектура сайта построена на статической генерации (Astro.js), и вместо традиционной админ-панели (GUI) этот проект реализует концепцию **LUI (Language User Interface)**. В качестве администратора выступает ИИ-агент на базе **Gemini CLI**.

## Структура
* `skills/`: Директория с исполняемыми инструкциями для агента. Каждый навык подчиняется спецификации `designxdevelop/agent-skills`.
    * `manage-content/`: Навык для CRUD-операций с контентом (Markdown).

## Использование
Запустите `gemini-cli` внутри этой папки. Агент автоматически обнаружит навыки и будет готов принимать голосовые или текстовые команды типа:
> "Добавь в меню новую позицию: Раф-кофе, стоит 300 рублей, категория - напитки"
